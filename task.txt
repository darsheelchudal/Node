REST API - JSON

GET /api/users - List all users  
GET /users - List all users  

Dynamic Path parameters (we use colon which confirms that its dynamic)
GET /api/users/:id


GET /api/users/1 - Get the users with ID 1
GET /api/users/2 - Get the users with ID 2

POST /api/users - Create new user


PATCH /api/users/1 - Update  the existing users

DELETE /api/users/1 - Delete the User with ID 1
